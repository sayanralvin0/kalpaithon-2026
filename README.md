# ZeroWaste Hub ♻️🍱

## 📌 Project Overview & Features

ZeroWaste Hub is a web-based platform designed to reduce food waste and promote proper waste disposal.

It connects restaurants and users to:
- Redistribute leftover food at discounted prices
- Help users dispose waste correctly using image classification
- Track environmental impact

## 🔹 Key Features
- Food listing by restaurants
- User ordering system (with quantity selection)
- Dynamic quantity & time tracking
- Waste image upload and classification
- Impact tracking (food saved, waste processed)

---

## ⚙️ Tech Stack Used

- Frontend: HTML, CSS, JavaScript  
- Backend: Flask (Python)  
- Database: Firebase (Firestore)  
- Image Processing: Rule-based logic / OpenCV (optional)
