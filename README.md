# ZeroWaste Hub ♻️

## Problem Statement
Food wastage is a major issue in restaurants and households, while improper waste disposal contributes to environmental pollution. There is no unified system that addresses both food redistribution and responsible waste management.

- - - -

## Detailed Description of the Problem
This waste stems from overproduction, buffet excess, and plate waste, representing a significant portion of the total ~78 million tonnes of food wasted in India each year. At the same time, many people lack awareness or tools to properly segregate and dispose of waste, leading to environmental damage.

  Existing solutions focus either on food delivery or waste management separately, but fail to integrate both aspects into a single, practical system. This gap results in continued food wastage and inefficient waste handling.

- - - -

## Proposed Solution (Brief)
ZeroWaste Hub is a web-based platform that enables restaurants to list leftover food at discounted prices, allowing users to purchase it before it is wasted. After consumption, users can upload images of waste, and the system classifies it into appropriate categories (dry, wet, recyclable) and provides disposal guidance.

  The platform also tracks the overall impact by showing how much food has been saved and how much waste has been processed.

- - - -

## Key Features
- Food listing by restaurants  
- User ordering system with quantity selection  
- Dynamic quantity and time tracking  
- Waste image upload and classification  
- Impact tracking (food saved, waste processed)  

- - - -

## System Flow
Restaurant adds food → Food is listed → User orders → Quantity updates → User uploads waste → Waste is classified → Impact updated
